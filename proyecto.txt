El proyecto es una plataforma Ecommerce para Pymes donde las personas pueden visualizar los productos que se ofrecen y añadirlos a un carrito para poder realizar el proceso de compra. El usuario debe registrar e iniciar sesión para poder realizar compras en la plataforma. Las Pymes pueden crear una cuenta donde registran sus productos y tienen disponible una visualización de registros de las ventas de sus productos y el estado de su cuenta.
